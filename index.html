About

Kurrawong is a tiny Sematic Data company that supports a few cool tools for Semantic/Linked Data enthusiasts.

Tools

  • Interactive Timescale - an interactive version of the International Commission on Stratigraphy's chronostratigraphic chart.
  • pyLODE Online API - the in development API to run pyLODE.

About

Kurrawong is the close-to one-man show of Nicholas Car (ORCID, LinkedIn). It's been running on-and off for a decade and now exists to pick up Nick's Semantic Web/Linked Data/IT side projects, some of which are quite long-running.
